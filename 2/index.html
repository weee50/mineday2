NOTE: If the board looks invisible, simply click on it. It will generate like normal, and everything will be visible.

Today, there are single, triple, and pentuple (5) mines. They work as you would expect, counting as 1, 3, and 5 mines respectively.
Yes, you will have to double right click for a triple flag and triple right click for a quintuple flag. The alternative would be allowing double and quadruple flags to exist, which would both make getting quintuple flags take more time and allow useless double/quadruple flags to exist.
The fact that all of these numbers are odd means a few different logical deductions can be made. For example, if a number is even, there must be an even number of squares with some sort of mine around them, and a similar thing happens with odd numbers.
